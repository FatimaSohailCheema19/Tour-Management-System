# 🌍 Tour Management System (Database + Frontend)

This project presents the database design and a basic frontend interface of a Tour Management System, developed as part of the Database course.

---

## 📌 Project Overview

The system is designed to manage tours, customers, and bookings.
It includes both:

* 📊 Database design (ERD + Relational Schema)
* 🎨 Frontend interface for user interaction

---

## 🎯 Objectives

* Design a structured database using ERD
* Convert ERD into relational schema
* Develop a simple frontend interface
* Represent real-world tour management operations

---

## 🧩 Project Components

### 🗄️ Database Design

* Entity Relationship Diagram (ERD)
* Relational Schema

### 🎨 Frontend

* User-friendly interface
* Pages for tours, bookings, and customer interaction
